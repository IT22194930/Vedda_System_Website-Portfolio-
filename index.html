🔬 Research Domain

Research Domain

Understanding the challenges, gaps, and innovative approaches to preserving endangered indigenous heritage.

📚
00 - Literature Survey

Literature Survey

Prior research shows that cascaded translation pipelines are effective for multilingual systems, and Sinhala-English neural machine translation can provide a practical bridge for related low-resource languages such as Vedda [1], [2]. Studies on phonemic transcription also emphasize that languages without standardized writing systems need both manual and automated annotation support to make speech data usable at scale [3].

Work on Vedda preservation consistently points to documentation, digitization, and technology-enabled support as the core requirements for survival [4], [5]. Web platforms have improved access to indigenous language resources, but many still focus on content delivery rather than learner assistance [6]. In contrast, AI-based vocabulary learning research highlights the value of adaptive support and guided practice, which this project extends to Vedda language learning [7].

For digital heritage preservation, OAIS provides a robust archival model while CIDOC CRM supports interoperable cultural metadata and semantic description [8], [9]. Automated identification research further shows that deep learning and hybrid CNN-SVM methods can perform strongly on structured classification problems, making them suitable for artifact recognition workflows [10].

Speech visualization studies demonstrate that 3D articulatory feedback improves pronunciation learning by exposing mouth and tongue motion instead of relying only on text or audio [11]. Recent lip-sync and tongue-animation methods confirm that realistic motion improves intelligibility and naturalness [12], [13], and WebGL talking-head systems show that these ideas can run directly in browsers [14].

Research implication: the proposed platform combines browser-based 3D rendering, IPA-driven viseme control, translation support, archival metadata, and adaptive learner guidance to address the current gap in Vedda-centered intelligent support.

⚠️
01 - Problem

Research Problem

The Vedda language of Sri Lanka is critically endangered - an oral-only language spoken by fewer than 300 people, with no written script, formal grammar, or digital footprint. As elder speakers pass away, entire vocabularies, oral histories, and cultural knowledge are lost irreversibly.

Without immediate intervention through modern digital preservation technologies, this unique linguistic and cultural identity faces complete extinction within a single generation.

🔍
02 - Gap

Research Gap

Current preservation efforts rely on static text-based dictionaries that fundamentally fail to capture the multi-dimensional nature of an oral language.

  • Existing tools cannot perform real-time Vedda-to-Sinhala speech translation
  • No AI model exists for identifying and classifying Vedda cultural artifacts
  • No 3D phonetic visualization exists for Vedda speech sounds
  • No gamified learning platform engages younger generations in language preservation
🎯
03 - Objectives

Research Objectives

  • 3D Phonetic Learning Model - Build interactive 3D visualizations of mouth, tongue, and throat positions for accurate Vedda pronunciation learning
  • Real-time Speech Translation - Develop a fine-tuned Whisper-based model for Vedda ↔ Sinhala real-time speech translation and transcription
  • AI Artifact Identification - Create a Hybrid CNN-SVM classifier for automated recognition and documentation of Vedda cultural artifacts
  • Gamified Learning Platform - Design an engaging, game-based system to teach Vedda language and culture to younger generations
⚙️
04 - Methodology

Methodology

Our system employs a microservices architecture integrating four specialized AI/ML components:

  • Hybrid CNN-SVM Model - Combines deep feature extraction (CNN) with robust classification (SVM) for artifact identification with high accuracy
  • Fine-tuned Whisper Model - OpenAI's Whisper adapted for low-resource Vedda speech recognition and translation
  • 3D Phonetic Engine - Real-time articulation modelling using Three.js for interactive mouth position visualization
  • Gamification Framework - Points, levels, and cultural challenges layered over language lessons to drive engagement
🛠️
Technologies Used

Technologies Used

These are the technologies used across the project modules for vocabulary learning, speech translation, digital preservation, and 3D pronunciation learning.

  • Web Technologies - React.js, HTML, CSS, JavaScript, Node.js, and Express for the interactive web applications and APIs
  • Databases - MongoDB, Firebase, and SQLite for user data, learning progress, and preservation records
  • AI / ML Frameworks - PyTorch, Hugging Face Transformers, and Keras for speech, translation, and classification workflows
  • Artifact Identification - CNN (Convolutional Neural Network) + SVM (Support Vector Machine) hybrid model for artifact identification and classification
  • Speech and Translation - Whisper, OpenNMT, MarianMT, Coqui TTS, TensorFlowTTS, Google Speech-to-Text, and Web Speech API
  • Adaptive Learning - SM-2 spaced repetition, Playlyfe, and GamifyJS for vocabulary practice and gamification
  • 3D and Media - Three.js, WebGL, Blender, Mixamo, GSAP, and Web Audio API for 3D phonetic and audio-visual learning
  • Speech and Phonetics - IPA-driven viseme control and custom phoneme sets for Vedda pronunciation support
  • Deployment - GitHub, Vercel, and Azure for source control and hosting

System Architecture

A modular microservices layout connecting the client, gateway, AI services, data stores, and external APIs into one preservation platform.
